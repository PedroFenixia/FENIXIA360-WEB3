# FENIX IA - Landing Page WEB3

## Proyecto
Landing page para **FENIX IA 360**, plataforma B2B SaaS de analítica de comunicaciones con IA.
Esta es la **variante WEB3** (tema claro, enfoque técnico/minimalista).

## Estructura
- `index.html` - Página completa (HTML + CSS inline + JS inline)
- `logo.png` - Logo de FENIX IA
- `CNAME` - Dominio: web3.fenixia360.com

## Stack Técnico
- HTML5, CSS3 (inline), Vanilla JavaScript
- Google Fonts: Inter + JetBrains Mono
- Sin frameworks, sin build tools, sin dependencias
- Deploy: GitHub Pages

## Diseño WEB3
- **Tema**: Claro (#ffffff) con acentos azul (#4d65ff)
- **Enfoque**: Técnico y minimalista - estética terminal/developer
- **Audiencia**: Perfiles técnicos, CTOs, desarrolladores
- **Diferenciadores**: Casos de uso estilo log/terminal, diseño limpio

## Producto FENIX IA
- **FENIX Pulse**: Analítica IA (transcripción, sentimiento, oportunidades) - €49/mes
- **FENIX Connect**: Hub de comunicaciones unificado - €39/mes
- **FENIX 360**: Bundle completo (Pulse + Connect) - €79/mes
- **Canales**: Llamadas, Email, WhatsApp, Telegram, Meta
- **Sectores**: Clínicas, Inmobiliarias, E-commerce, Asesorías, Seguros

## Variantes Hermanas
- WEB1 (web1.fenixia360.com): Tema oscuro, storytelling emocional, púrpura (#7c3aed)
- WEB2 (web2.fenixia360.com): Tema claro, enfoque ROI, teal (#0d9488)

## Reglas de Estilo
- Todo el contenido en **español**
- Mantener consistencia de precios y features entre las 3 variantes
- Responsive: breakpoints en 900px y 600px
- Mantener estética técnica/terminal en casos de uso
- No añadir dependencias externas innecesarias
